# java.util.List

- list is interface
- used to store ordered collection of elements
- maintains order of elements in which they are added
- allows duplicates
- implementation of List interface is **ArrayList**, **Stack**, **Vector**, **LinkedList**


```java

List<Integer> l = new ArrayList<>();

// add 
l.add(1);
l.add(2);

// get
l.get(0);

```
